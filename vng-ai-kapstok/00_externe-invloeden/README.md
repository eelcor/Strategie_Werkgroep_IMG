# Externe invloeden

Deze laag verzamelt ontwikkelingen van buiten de gemeente die de AI-strategie voeden. De werkgroep duidt per thema wat de betekenis is voor gemeenten.

Gemeenten en werkgroepleden kunnen hier documenten, rapporten, beleidsnotities en signalerende stukken delen.

---

## Submappen

| Map | Inhoud |
|---|---|
| [maatschappelijke-ontwikkelingen](./maatschappelijke-ontwikkelingen/) | Publiek debat, onderzoeksrapporten, ethische vraagstukken, burgervertrouwen |
| [overheid-wetgeving](./overheid-wetgeving/) | Nederlandse Digitaliseringsstrategie, coalitieakkoord, AI-act, BZK-beleid |
| [vng-ontwikkelingen](./vng-ontwikkelingen/) | VNG-resoluties, GEMMA-updates, werkgroepagenda's, ledenraadbesluiten |

---

## Hoe upload ik een document?

1. Ga naar de juiste submap
2. Klik **Add file → Upload files**
3. Sleep je bestand naar het scherm (PDF, DOCX of MD)
4. Geef bij "Commit message" een korte beschrijving, bijv.:  
   `Toevoegen: WRR-rapport Opgave AI (samenvatting relevante passages)`
5. Kies: **commit direct op `main`** (gepubliceerd) of **maak een nieuwe branch** (concept)

---

## Bestandsnaamconventie

Gebruik een herkenbare naam zonder spaties:

```
JJJJ-MM_Organisatie_Titel.pdf
```

Voorbeelden:
- `2025-01_WRR_Opgave-AI-samenvatting.pdf`
- `2025-03_IO-Research_AI-vertrouwen-gemeenten.pdf`
- `2024-11_Rijksoverheid_NDS-gemeenterelevante-passages.pdf`
