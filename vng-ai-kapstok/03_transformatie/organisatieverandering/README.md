# Organisatieverandering — programmatische aanpak

## Werkgroep beschouwing

> AI-transformatie is geen IT-project maar een organisatieprogramma. Gedragsverandering, nieuwe rollen en heldere mandaten moeten parallel lopen aan technische implementatie — niet erna.
>
> De programmatische aanpak betekent: een eigen programmastructuur naast de lijn, gefaseerde doelen op drie niveaus (bewustwording → inbedding → verankering), en koppeling aan de volwassenheidsscan als meting.
