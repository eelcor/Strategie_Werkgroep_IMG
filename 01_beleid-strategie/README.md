## Hoe levert uw gemeente een bijdrage?

Een bijdrage kan zijn:
- Een beleidsstuk of notitie die u heeft opgesteld
- Een praktijkervaring of casus
- Een vraag of knelpunt dat u bent tegengekomen
- Een toetsingskader of checklist die u heeft ontwikkeld

**Stap 1:** Ga naar de juiste themamap  
**Stap 2:** Maak een submap aan met uw gemeentenaam: `gemeente-amsterdam/`  
**Stap 3:** Upload uw document(en) in die map  
**Stap 4:** Pas de README van de themamap aan: voeg een rij toe in de bijdragentabel  
**Stap 5:** Maak een Pull Request aan zodat de werkgroep uw bijdrage kan beoordelen
