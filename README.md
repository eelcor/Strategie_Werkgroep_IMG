## Hoe werkt het?

### Documenten uploaden (geen Git-kennis nodig)
1. Navigeer naar de juiste map
2. Klik op **Add file → Upload files**
3. Sleep je bestand (PDF, DOCX, MD) naar het scherm
4. Vul een korte beschrijving in bij "Commit changes"
5. Klik op **Commit changes**

### Concept vs. gepubliceerd
- **`main`-branch** = openbaar, goedgekeurd
- **`concept`-branch** = besloten werkruimte voor de werkgroep

### Bijdrage leveren als gemeente
1. Upload je document in de juiste map op de `concept`-branch
2. Maak een **Pull Request** aan met een korte toelichting
3. De werkgroep beoordeelt en voegt zo nodig een beschouwing toe
4. Na goedkeuring wordt het samengevoegd in `main`
