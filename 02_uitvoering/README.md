# Uitvoering

De vier uitvoeringslagen van de kapstok. Hier delen gemeenten ervaringen, werkwijzen en documenten rondom de praktische inzet van AI.

## Lagen

| Map | Laag | Inhoud |
|---|---|---|
| [mens-organisatie](./mens-organisatie/) | Mens en organisatie | Rollen, competenties, cultuur, SPP |
| [proces-structuur](./proces-structuur/) | Proces en structuur | Kwaliteitstraat, ontwikkelproces, multidisciplinair werken |
| [technologie](./technologie/) | Technologie | AI-platforms, productlijnen, architectuurkeuzes |
| [data](./data/) | Data | Databeheer, kwaliteit, governance, soevereiniteit |

## Bijdrage leveren

Ga naar de juiste laag, maak een submap `gemeente-uwgemeente/` aan en upload uw document. Voeg daarna een rij toe aan de bijdragentabel in de README van die laag.
